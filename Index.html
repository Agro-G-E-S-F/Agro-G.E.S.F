Sobre o AGRO G.E.S.F

O AGRO G.E.S.F é um projeto que busca criar uma solução viável para pequenos agricultores que querem fazer uma identificação de pragas mais precisa em suas plantações em linha, o que reduz a utilização desnecessária de agrotóxicos que prejudicam tanto a saúde do produtor quanto o seu produto.

Nossos Objetivos
Temos os seguintes objetivos com o AGRO G.E.S.F

Precisão

Maior precisão possível para evitar falsos positivos

Agilidade

Detecção o mais rápido possível para o tratamento precoce

Acessível

Criar um produto acessível para pequenos produtores

Entenda Melhor Nosso Produto

O AGRO G.E.S.F é um dispositivo capaz de percorrer por lavouras de plantas rasteiras em todos os cenários possíveis identificando sinais pragas e doenças nas folhas.

As informações da análises são enviadas para um software desktop de acesso ao agricultor onde ele consegue ter acesso a dados como nome, foto e descrição do que foi identificado. Além disso o sistema possui uma página com gráficos de detecções por meses e um glossário onde mostra uma breve introdução de todas as pragas que já foram identificadas.
